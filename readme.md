ရည္ရြယ္ခ်က္
---------

ကြ်န္ေတာ္တို႕ အခုမွစၿပီး ဘာသာျပန္ေနတုံးပါ၊ ရည္ရြယ္ခ်က္ ကေတာ႕ Laravel PHP Framework Documencation ကို ျမန္မာလို ဖတ္ၿပီး Laravel Framework ကို အလြယ္တကူ ေလ႕လာႏိုင္ဖို႕ပါ၊

ဒီဘာသာျပန္မွုမွာ ဘယ္လို ဘာဝင္ကူညီႏိုင္လဲ
--------------------

ဟုတ္ကဲ႕ ကြ်န္ေတာ္တို႕လည္းအလုပ္နဲ႕ပါ အားတဲ႕အခ်ိန္ေလးဘာသာျပန္ၿပီးေတာ႕ဒီမွာတင္တာပါ။ တကယ္လို႕ကြ်န္ေတာ္တို႕ကို ကူညီၿပီးဘာသာျပန္ခ်င္တယ္ဆိုရင္ဒီ Repo မွာ pull request လုပ္ၿပီး contribute လုပ္ႏိုင္ပါတယ္။ ကြ်န္ေတာ္တို႕ဘာသာျပန္ၿပီးသားေတြကိုလည္း စာလံုးေပါင္းအမွားေတြကို git issue ေတြေပးႏိုင္ပါတယ္။ [Contribute Guide](contributing.md) ကိုဖတ္ေပးပါ။
